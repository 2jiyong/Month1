# 화성 달력 시계
## 설계과정 
1화성년 = 668화성일 (2년마다 윤년)
1주일 = 7화성일
1화성월 = 28화성일(6월마다 27일로)
1화성년 = 24화성월 

1. java.time.LocalDate와 java.time.temporal.ChronoUnit을 사용하여 입력받은 지구력이 며칠째인지를 알아낸다.  
2. 그 날짜를 화성력으로 변환한다.
3. 화성력으로 변환한 화성월의 달력을 출력한다.
